# 不用上b站也可以看弹幕

## 介绍

一个可以在任何视频网站上加载弹幕的浏览器扩展

来自 <span class="c-blue">Mr-Quin</span> 大佬的分享（名称是 {% badge danmaku-anywhere blue %}）

仓库地址：[Mr-Quin/danmaku-anywhere](https://github.com/Mr-Quin/danmaku-anywhere)

{% alertBlockquote info %}
项目配置比较简单，对于像我这种没钱又想要弹幕的比较合适
{% endalertBlockquote %}

![](https://tzyy-1330068502.cos.ap-beijing.myqcloud.com/665200d08d44f05d5bda654cd6144bb.png)

b站可以跟随大佬视频学习配置（b站名称：磷化镓）

![](https://tzyy-1330068502.cos.ap-beijing.myqcloud.com/c4707bc1086757d98df1f4573c9b088.jpg)

## 教程

本次教程以磷化镓大佬的视频为主，用edge浏览器和AGE动漫网

部分操作需要科学上网

AGE动漫网：[AGE动漫](https://m.agedm.org/#/)

{% details 下载插件 %}
点击Mr-Quin大佬的danmaku-anywhere仓库拉到最下面

#### 点击商店安装

![](https://tzyy-1330068502.cos.ap-beijing.myqcloud.com/9cfc7a4daca2e700c06de7d285a31c2.png)

然后下载插件（edge浏览器下载提示的那个）

![](https://tzyy-1330068502.cos.ap-beijing.myqcloud.com/f456db2f08e5c2cf6a7196d7070f1b8.png)

#### 右上角出现扩展

![](https://tzyy-1330068502.cos.ap-beijing.myqcloud.com/957c13a2d6ec56e3fc9a250fe3536cb.png)
{% enddetails %}

{% details 开始配置 %}
#### 打开AGE动漫网

选择一个想看的动漫，点进去播放视频

#### 右键点击统计信息

![](https://tzyy-1330068502.cos.ap-beijing.myqcloud.com/f8611a5eb91ed99a26059b967973ed5.png)

#### 复制视频网址

![](https://tzyy-1330068502.cos.ap-beijing.myqcloud.com/a1035feadac9a4880b1744cddc62100.png)

然后在一个新的页面输入刚刚复制的网址打开之后

#### 打开插件

![](https://tzyy-1330068502.cos.ap-beijing.myqcloud.com/b615e5a610afd0e9e0d5490bddee8d5.png)

#### 复制要用的网址

![](https://tzyy-1330068502.cos.ap-beijing.myqcloud.com/974ceb886150bde596c3c4bc79fff4b.jpg)

#### 按步骤进行配置

![](https://tzyy-1330068502.cos.ap-beijing.myqcloud.com/2e4a4eb9bc31210ba8ce9381792168b.jpg)

![](https://tzyy-1330068502.cos.ap-beijing.myqcloud.com/eb1cf5fc3b0b563f26bf9238c532f63.jpg)

#### 选择需要的弹幕

![](https://tzyy-1330068502.cos.ap-beijing.myqcloud.com/eace83e477a6b8f487630ec448fb499.jpg)
