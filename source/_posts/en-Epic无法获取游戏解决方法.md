# Fixing "Cannot Claim More Free Games" on Epic

## Epic Platform

Epic Games Store: [https://store.epicgames.com/](https://store.epicgames.com/)

Epic released "Ghostrunner 2" for free — I rushed to claim it but hit an error.

## Solution

### 1. Network Issues
Refresh your connection or switch networks.

### 2. Proxy / VPN
These platforms often require VPN. Make sure yours is enabled.

{% alertBlockquote danger 3. Never Use Campus Network! %}
This was my exact problem. Campus networks block these connections. Switch to mobile hotspot or home WiFi.
{% endalertBlockquote %}

### 4. Use the Browser Store
Paste the Epic Store URL into your browser to claim directly from the website.

{% alertBlockquote info %}
These are all the methods I found and tested.
{% endalertBlockquote %}
