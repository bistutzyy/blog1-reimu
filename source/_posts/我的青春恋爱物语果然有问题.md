> <span class="c-red">青春是一场谎言、一种罪恶。</span>
> ——比企谷八幡


{% alertBlockquote info %}
千叶县立总武高等学校，一个以「青春」为名的战场。在这里，每个人都在上演一场名为「现充」的戏码，直到一位眼神死透的少年用最冰冷的口吻宣告：<span class="c-red">青春是一场谎言、一种罪恶。</span>
{% endalertBlockquote %}

{% badge 比企谷八幡 blue %}，重度死鱼眼、社交恐惧、对所谓「青春」嗤之以鼻的高二学生。因性格过于扭曲而被班主任平冢静强行塞进了一个神秘社团——<span class="c-purple">侍奉部</span>。在那里，他遇见了完美到不真实的 {% badge 雪之下雪乃 purple %}，以及元气过剩的 {% badge 由比滨结衣 pink %}。

<span class="c-cyan">侍奉部</span>的宗旨很简单：帮助有困难的同学。但八幡从不提供温暖的安慰，他只给出最残酷、最彻底、最不留情面的解法。他用 <span class="c-red">自爆式</span>的解决方式，一次又一次将自己推向孤立，换取他人的圆满结局。

{% badge 雪之下雪乃 purple %} 这样评价他：

> <span class="c-cyan">比企谷君，你的做法从一开始就是错的。你以为牺牲自己就能解决问题——但那不是帮助别人，那只是在伤害你自己。</span>

{% badge 由比滨结衣 pink %} 则用另一种方式注视着他：


> <span class="c-pink">小企，其实是个比谁都温柔的人。只是温柔的方式，笨拙到了让人心疼。</span>


侍奉部的日常里，有彩虹面包的甜，有烟花大会的灿烂，有文化祭的喧嚣，也有一言不合就互怼的毒舌对决。八幡与雪乃的每一次对话都是棋逢对手的哲学辩论——

> <span class="c-green">世界不会因为你流泪就改变什么。但如果你摆出一副'我哭过了'的表情，就会有人来帮你擦眼泪——不过那只是骗取同情的另一种手段罢了。</span>
> ——比企谷八幡


> <span class="c-red">说到底，人是无法真正理解彼此的。所谓理解，不过是在误解与误解之间，找到一个可以共存的缝隙罢了。</span>
> ——比企谷八幡


{% alertBlockquote tip SUCCESS %}
《我的青春恋爱物语果然有问题》之所以能从千万轻小说中脱颖而出，成为唯一的 <span class="c-red">殿堂级轻小说三连冠</span>（史无前例地连续三年蝉联「这本轻小说真厉害！」榜首），正是因为它撕开了「青春」那一层虚假的糖衣。

它不是告诉你青春有多美好，而是告诉你：<span class="c-green">青春可以不美好，人生可以不顺利，性格可以有缺陷——但这并不妨碍你继续走下去。</span>
{% endalertBlockquote %}

这也是为什么，八幡的每一句毒舌都像在替我们说出了那些不敢承认的心里话：


> <span class="c-orange">努力一定会得到回报？那只是谎言。但若不努力，连获得回报的可能性都会消失。</span>
> ——比企谷八幡


> <span class="c-cyan">破碎的东西无法恢复到原样。但我们可以拾起碎片，重新塑造出新的形态。</span>
> ——比企谷八幡


> <span class="c-purple">孤独并非恶事。倒不如说，正因为深知孤独为何物，才能体会他人的痛苦。</span>
> ——比企谷八幡


> <span class="c-red">真正的强大，不是一个人独自站立。而是即使有人扶持，依然能用自己的双脚稳稳地站立。</span>
> ——比企谷八幡


> <span class="c-green">能够改变的只有自己。不要妄图去改变他人。但当你自己改变时，看待世界的方式也会随之改变。</span>
> ——比企谷八幡


最终，在经历了无数次擦肩而过与青涩试探之后，八幡终于给出了属于他的答案——那不再是冰冷的自爆与回避，而是笨拙却坚定的一声：

> <span class="c-cyan">我想要的，是真物。不是虚假的，而是真正的羁绊。</span>
> ——比企谷八幡


{% alertBlockquote tip %}
这就是春物。一部以 <span class="c-red">毒舌</span>为武器、以 <span class="c-blue">孤独</span>为铠甲、以 <span class="c-purple">真心</span>为赌注的青春物语——第一部，也是唯一一部，真正配得上「<span class="c-red">殿堂</span>」二字的轻小说。
{% endalertBlockquote %}
