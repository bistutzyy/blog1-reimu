## 她是哔哩哔哩

你知道吗？**B站的全名"bilibili"，就是来自于她。**

<span class="c-blue">御坂美琴</span>，学园都市七位 Level 5 超能力者中排名第三——代号 <span class="c-red">「超电磁炮」</span>。她拥有操纵电力的能力，指尖跃动的电光，是这座科学之城最耀眼的一道风景。而在粉丝心中，她有一个更亲切的名字：<span class="c-red">炮姐</span>。

B站投币、充电——这些刻入 DNA 的操作，其源头正是炮姐的招牌技能。一个角色能成为一个平台的灵魂，这本身就是传奇。

{% alertBlockquote tip SUCCESS %}
她是轻小说史上无可争议的人气女王。「这本轻小说真厉害！」女性角色部门 **9 次登顶**（2010-2020 年间），2011 年斩获国际最萌萌王，萌战冠军累计超过 55 次。
{% endalertBlockquote %}

## 常盘台的王牌

{% badge 御坂美琴 blue %}，常盘台中学二年级，学生寮 208 号室。被全校女生仰慕地称为「常盘台的 Ace」。

她的招牌技能：
- <span class="c-red">超电磁炮</span>：以游戏厅的金属代币为弹丸，三倍音速射出——硬币划过空气时会因摩擦熔化，化作一道橘色光束
- <span class="c-cyan">雷击之枪</span>：从刘海间放出的蓝白色电击，光速必中
- <span class="c-purple">铁砂之剑</span>：利用磁力操控铁砂，化为可变形的利刃或风暴
- <span class="c-green">电磁骇客</span>：能轻易破解学园都市的电子系统，甚至直接入侵卫星

但她真正的强大不止于此。在生死关头，美琴会召唤落雷——最大十亿伏特的闪电从天而降，那画面足以成为一代观众心中永不褪色的记忆。

{% alertBlockquote danger %}
**Level 6 绝对能力者计划：** 用两万名"妹妹们"与一方通行对战 20000 次，让一方通行进化至 Level 6。美琴得知真相后，独自闯入实验室，试图以牺牲自己的方式终结这场屠杀。一个 14 岁的少女，选择独自背负两万条人命——这是《超炮》系列最沉重、也最动人的篇章。
{% endalertBlockquote %}

## 不是完美的电击公主

美琴是个矛盾体。

她强大到能与一方通行对峙，却会在自动贩卖机吞币时暴躁地使出电击回旋踢；她被称为「最强无敌的电击公主」，却会在便利店里站着看漫画，痴迷于呱太这种幼儿园级别的吉祥物；她嘴上说着"那个笨蛋"——却会在上条当麻遇到麻烦时第一个冲过去。

<span class="c-purple">傲娇</span> 这个词，几乎就是为她量身定做的。

她穿安全裤、不在乎打扮，却对自己的贫乳暗中在意；她爽朗仗义像个男孩子，却在对当麻偶尔的温柔中暴露出不设防的少女心。这种 <span class="c-cyan">反差萌</span>，正是她跨越二十年依然让人无法抗拒的原因。

{% blockquote 御坂美琴 %}
即使这样，我还是想守护这个世界。
{% endblockquote %}

## 她身边的人

{% alertBlockquote info %}
**白井黑子** —— 常盘台的瞬间移动能力者，美琴的室友。对美琴的称呼永远是"姐姐大人"，痴汉属性爆棚，但关键时刻是最可靠的战友。可惜她的感情，终究是"届不到的悲剧"。

**初春饰利 & 佐天泪子** —— 栅川中学的 Level 1 风纪委员和 Level 0 无能力者。四个女孩凑在一起时，是《超炮》最温暖的部分——吃甜品、逛街、卷入事件然后解决它。

**食蜂操祈** —— Level 5 第五位，心理掌握。常盘台真正的"女王"，与美琴的关系是欢喜冤家兼亦敌亦友。两人在《超炮 T》中的互动被粉丝津津乐道。
{% endalertBlockquote %}

## 写在最后

御坂美琴让我着迷的，不是她有多强。

是她明明可以冷眼旁观，却偏要为一个素未谋面的克隆人赌上性命；是她明明可以活在"常盘台王牌"的光环里，却愿意穿着幼稚的呱太睡衣在宿舍里打游戏；是她被击倒了无数次，依然会从废墟里爬起来，在刘海间迸出那一缕蓝白色的电光。

{% alertBlockquote tip %}
**"你指尖跃动的电光，是我此生不变的信仰，唯我超电磁炮永世长存。"**

这句 B站弹幕里反复出现的应援辞，或许就是所有炮厨的心声。
{% endalertBlockquote %}

二十年来，她始终站在那里——学园都市的天台上，指尖转动着一枚游戏厅的硬币。风扬起她的短发和常盘台的校服裙摆，身后是无尽的钢铁森林与夕阳。

她是御坂美琴。她是炮姐。她是哔哩哔哩。

{% alertBlockquote info %}
**她是这个科学世界里，最温暖的一道电流。**
{% endalertBlockquote %}
