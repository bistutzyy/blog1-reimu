# "只有受到眷顾的人，才能毫不掩饰"

## 关于这部番

《{% badge 辉夜大小姐想让我告白 blue %}》——B站校园恋爱番热度 No.1，三季豆瓣评分均在 <span class="c-red">9.0+</span>，被广泛誉为"近十年恋爱喜剧天花板"。两个互相暗恋的天才，因为"先告白的人就输了"这种傲娇心理，展开一场场沙雕又甜蜜的恋爱头脑战。

但剥开沙雕的外壳，这其实是一部关于 <span class="c-blue">努力、自卑与勇气</span> 的成长故事。而站在这个故事中心的，是一个把手伸向月球的少年——

## 他不是天才

{% badge 白银御行 blue %}，秀知院学园学生会会长，全国模拟考第一，品学兼优、行事模范。

但这些光环，全是他用命拼来的。

他并非天生聪颖。小时候成绩不理想，父亲公司破产，母亲离家出走——而且只带走了妹妹。他被母亲判定为"没有才能的孩子"。从那以后，他每天学习超过 10 小时，同时打工补贴家用。他的房间里贴满了标语：

{% alertBlockquote danger %}
<span class="c-red">"努力是最强的才能"</span>
<span class="c-red">"为了成为站在四宫身边的男人"</span>
{% endalertBlockquote %}

用光的圆珠笔堆积如山。睡眠严重不足，才有了那双让周围人畏惧的凶恶眼神。他**不是天才，是努力成瘾者**——靠超人的毅力考入贵族学校，并长期保持年级第一。

<p><span class="c-blue">白银御行</span>，最喜欢的男主之一。喜欢他，不是因为他是完美的学生会长，而是因为他那 <span class="c-purple">骨子里的自卑与不服输</span>——明明是个平凡人，偏要与天才站在同一高度。</p>

## 通往月亮的荆棘路

他曾说：

> <span class="c-purple">"若是我的话，就绝对不会放开辉夜公主的。我会登上月亮将她夺回，无论花上几十年、几百年。若这是我们的故事，我要将那份话语的深意全部看穿，那就不会是那样的结局了吧。"</span>

他知晓辉夜姬终会像传说中那般离开地面，飞向高不可攀的月亮——就如同四宫辉夜终将离开校园，走向遥不可及的世界。可他偏要与之抗衡。

那贴满了标语的房间和堆积如山的圆珠笔，都只为了能让自己 <span class="c-red">"成为站在四宫辉夜身边的男人"</span>。这是一场以努力为武器、以爱意为赌注的战争。

世界上有两种人：<span class="c-green">受到眷顾的人，和拼命追赶的人</span>。辉夜属于前者，而白银属于后者。他没有血统，没有天赋，没有财富——他只有日复一日的努力，和一份不愿放弃的初恋。

## 行动派的浪漫

他还曾讲过：

> <span class="c-cyan">"我不是说了有想让你看的东西吗，因为我一直没法妥协，因为我一直难以启齿，所以我只能用我能准备到的最棒的礼物送给你，你能懂吧，四宫。这就是，我对你的感情。"</span>

他是行动派。烟火大会时，孤独的辉夜流露出对烟花的渴望，他立马出现：

> <span class="c-green">"那我就让你看看吧，来吧四宫，你要是那么想看的话，就跟我来吧。忍耐了这么久，辛苦你了，你一点都不奇怪。"</span>

而后带着她在 20 分钟内从东京赶到千叶，只为了让她实现看烟花的梦想。

奉心祭上，他以怪盗亚森的身份放出漫天心形气球——那是整部作品中最令人心动的名场面。他骨子里是个不接受悲剧、只追求完美浪漫的 {% badge 理想主义者 red %}。

## 卑微到尘埃里，开出花来

他也曾袒露心声——

{% alertBlockquote danger %}
<span class="c-red">"比起被喜欢，我更想成为被需要的人。"</span>
{% endalertBlockquote %}

这句话，是他整个人的底色。

白银内心深处背负着童年创伤——被母亲抛弃、被判定"没有才能"。他形成了"必须不断努力才能获得他人认同"的病态心理。在辉夜面前，他害怕暴露真实的自己，担心一旦告白被拒就会失去一切。

喜欢一个人，会卑微到尘埃里，然后开出花来。**平民精英**的身份让他打破"门当户对"的传统叙事——他不是天生的天才，而是用汗水拼出一条通往月亮的荆棘之路的少年。

{% alertBlockquote tip SUCCESS %}
正如漫画标题《竹取物语》的大伴御行原型一样——明知挑战辉夜姬是不可能完成的任务，他依然选择正面迎战。这份勇气与执着，正是他最大的魅力。
{% endalertBlockquote %}

## 写在最后

我很钦佩他，但我成为不了他。

不是每个人都拥有这份极致的自律与勇气。但至少，在他的故事里，我们看到了一个普通人的可能性——即使生来不被眷顾，也可以靠双手触及星辰。

他是会长，是怪盗亚森，是白色巨蟒。他是一个把努力变成了超能力的普通人。他是——

{% alertBlockquote info %}
<span class="c-blue">只有受到眷顾的人，才能毫不掩饰。而白银御行，是那个不被眷顾，却偏要站到月亮上的人。</span>
{% endalertBlockquote %}
