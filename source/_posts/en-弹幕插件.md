# Danmaku Without Bilibili

## Introduction

A browser extension that loads bullet comments on any video site. By <span class="c-blue">Mr-Quin</span> — {% badge danmaku-anywhere blue %}.

Repo: [Mr-Quin/danmaku-anywhere](https://github.com/Mr-Quin/danmaku-anywhere)

{% alertBlockquote info %}
Simple setup. No Bilibili membership needed.
{% endalertBlockquote %}

## Tutorial

{% details Install the Extension %}
Download from browser store after visiting the repository.
{% enddetails %}

{% details Configuration %}
Open AGE Anime site, play a video, copy the video URL, paste into the extension, and choose your danmaku source.
{% enddetails %}
