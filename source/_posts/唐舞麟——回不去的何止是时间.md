## 为什么喜欢唐舞麟

第一次看《斗罗大陆3龙王传说》是在2012年，当时龙王传说已经完结了，终极斗罗也开始更新了。不过是我初次遇到舞麟，相比之前斗一和斗二，这是我唯一完整看完的（在2014年看完的），也是我最喜欢的一部。而唐舞麟也是我最喜欢的小说男主。

之所以喜欢 <span class="c-blue">50</span>（舞麟的谐音），更多的是他的个人魅力吧。他从普通孩子成长为一代强者，中间充满了挑战和机遇，坚韧不拔，不断挑战自我。

当然他也是最难的一位主角——年少与父母分离，因为武魂是 <span class="c-green">蓝银草</span> 的问题前期修炼一直落后伙伴，因为 <span class="c-red">金龙王血脉</span> 的问题承受很多痛苦。从 **3 级废武魂** 到东海学院 0 班，到史莱克学院，再到史莱克七怪——最后成为 {% badge 史莱克海神阁阁主 blue %}、{% badge 唐门门主 blue %}、{% badge 九级神匠 red %}。

{% alertBlockquote danger %}
中间遇到过太多挫折：因魂兽与人类的矛盾与爱人为敌，圣灵教与史莱克的矛盾，联邦与史莱克的立场不同，与战神殿的竞争，位面的袭击，传灵塔高层的问题，史莱克被毁重建，老师离世，海神阁阁主夫妇离世，养父母被害，<span class="c-red">千日葬龙</span>……太多太多的苦，50 一个人走了下来，直到成为人类第一强者。最后的最后，他选择了与古月娜殉情。
{% endalertBlockquote %}

斗三的我是真的喜欢，无论是在东海学院的那段懵懂上进的日子，还是在史莱克拼命修炼的时光。斗三的七怪每个我都很喜欢，他们像是活过来了，有血有肉。<span class="c-red">千日葬龙</span>，让我真的觉得很震撼。我看斗四，除了看麟月，还为了看东海小队，为了斗三七怪的重逢。

我最喜欢50，不是因为他最强，是因为他的经历让我感动，他的结局也是最可怜的。他为了和他的爱人在一起，不惜一切放弃了所有，他原本可以选择更美好的未来，但他没有，他依然爱的那个女孩，他放弃了当神的机会，他放弃了所有，却没有放弃他的爱人，直到冰封的的最后一刻，他终于和他的爱人在一起了!他不但没有后悔，反而笑了，发自内心的笑!

## 天霜龙冰——舞长空

<span class="c-blue">50</span> 的老师，最好的引路人。

这对师徒是那么的像——从阳光开朗到郁郁寡欢，痛失所爱永冻冰封。<span class="c-red">50 终究是步了舞长空的后尘。</span>

唐舞麟是舞长空最优秀的一个弟子，也是最像舞长空的一个弟子。50 终究还是成为了舞长空。

最感动的还是在东海学院时，舞长空看 50 放假一个人孤单，便带他去了另一个大陆——吃烤鱼，去唐门，祭奠舞老师逝去的爱人。

{% blockquote %}
哪怕直到最后一刻，他的眼眸之中，也只有眷恋与宠溺。在那寒冰的世界中，他的眼神却再无冰冷，也无痛苦。唯有那充满了爱恋的气息。

这一刻，<span class="c-cyan">天霜龙冰永久定格，永久凝固。</span>

唐舞麟一直强行控制的情绪终究无法忍耐，泪水瞬间而下。
{% endblockquote %}

自从第一次见到舞长空，舞老师就一直都是他的偶像——<span class="c-cyan">白衣蓝剑、天冰雪寒舞长空</span>。他更是带他出道，带他一步步迈向强者，走进史莱克学院的舞长空。

{% alertBlockquote info %}
舞长空，史莱克学院外院院长。我的老师。从第一天见到他的时候，他就是我的偶像。白衣蓝剑、冰天雪寒。他的一生，贡献给了史莱克，他的心，却留给了怀中的至爱。

**天霜**，是他的武魂。**龙冰**，是他的爱人。

愿您在天国能与师母重逢，天国不再有寒冷，唯有爱恋。
如果寒冰真的能够冻结一切，希望你们的爱情恒古长存。

落款是：**弟子，唐舞麟！**

黄金龙枪每一次颤动，都清晰地留下一个个字迹，而在这份字迹之中，充满了唐舞麟对舞长空的情感。
{% endalertBlockquote %}

可以说，唐舞麟与舞长空之间的感情已经不是纯粹的师生情——在他心目中，舞长空是 <span class="c-blue">老师</span>，也是 <span class="c-purple">兄长</span>，甚至还有一丝 <span class="c-red">父亲的影子</span>。

唐舞麟——
他是最好的伴侣，也是最佳的队长。

他对同伴说，**谁也不能带走你**。
史莱克城被毁的时候，他毫不犹豫地挡在了伙伴们身前；
挑选仙草的时候，他首先挑的是谁也不想要的；
他会鼓励队友，也会有腹黑的时候；
他对待感情非常利落，从不拖泥带水；
