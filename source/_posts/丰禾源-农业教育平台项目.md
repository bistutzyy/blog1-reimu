## <span class="c-green">项目背景</span>

随着农业现代化的推进，对农业从业者的知识和技能要求日益提高。然而，传统的农业教育已经无法满足广大农民及农业爱好者的学习需求。线上教育资源分散，缺乏针对农业领域系统且深度结合实践的平台；线下的农业培训又难以全面覆盖，并且成本较高。在此背景下，构建一个线上线下结合的农业教育平台显得尤为重要。

## <span class="c-red">市场痛点</span>

当前农业领域存在诸多信息不对称的问题。某菜农到外地参观学习种植技术时，看到有人种了上百公顷连片冬瓜，了解到其种植效益非常可观，回来就承包了10亩地种植冬瓜，在技术不到位、经验不足的情况下，首批冬瓜一公斤只卖几分钱，拉一车冬瓜到市场销售才卖300多元，除去车费所剩无几。当地还有很多冬瓜种植大户，几乎都血本无归——**盲目种植，比葫芦画瓢，是普遍现象。**

调查数据显示：多数农民购买种子或农药时并不关注品牌，主要依靠周围人推荐或市场占比，而市场上的产品品质却良莠不齐。对于大面积种植作物的家庭，<span class="c-red">70%以上</span>会采用机械化种植，大部分选择外租大型农具来完成耕作。

更值得关注的是粮食产后损失。根据中国农业大学武拉平教授团队的研究数据，农户粮食产后收获、储备和消费环节的总损失率达 <span class="c-red">11.39%</span>，其中水稻、玉米和小麦三种主粮作物的收获平均损失率为 <span class="c-orange">4.09%</span>。

## <span class="c-blue">解决方案</span>

项目打造了一个线上线下结合的农业教育平台——**丰禾源**。

**线上方面**，平台根据学员的基础、需求和学习进度，为其定制个性化的学习路径。利用电子信息技术整合农业高校、科研机构、企业等多方面资源，建设涵盖种植、养殖、农业机械等多领域的课程库，课程形式包括视频教程、图文资料、在线直播讲座等。平台还采用短视频、方言等接地气的方式进行教学，让中老年农户更容易接受和理解。

**线下方面**，团队深入走访调查不同地区农户面临的主要难题，联系各地专家传授知识，并积极向线上反馈，使平台体系更加完整。目前团队已经调查了多家农户的主要困难并为他们提供了科学指导，也与多名种植能手取得联系，积极整理各类资料。

{% details 技术架构 %}
- 利用 {% badge 关系型数据库 blue %}（MySQL、Oracle）存储用户信息、课程信息、教师信息及财务数据等，保障数据查询效率与安全性
- 利用 {% badge 非关系型数据库 green %}（MongoDB、Redis）存储用户笔记、讨论内容，以及缓存热门数据
- 以 <span class="c-red">"互联网+"</span>信息技术和电子商务技术为支撑运行
{% enddetails %}

{% alertBlockquote tip SUCCESS %}
相比于其他辅助农业的平台，丰禾源的 <span class="c-green">服务面更广</span>，产品收录采用用户上传方式，让用户之间可以互相分享经验、发现问题。采用分类方式，更便于搜索和寻找替代产品。在行业中更贴近农民生活，测评以各地区常见产品为主。
{% endalertBlockquote %}

## <span class="c-purple">创新意义</span>

- [x] 更灵活的学习方式 — 年轻人可主要依靠线上平台学习，年长者可通过线下学习，真正做到惠及全民
- [x] 覆盖范围大 — 线上线下联合教学可轻松覆盖全国各个区域，做到消耗小、成果高
- [x] 满足多样化需求 — 分模块、分类建立网站，方便学习者快速找到所需内容，还可通过互联网将农业与其他行业结合
- [x] 方言与普通话双版本 — 让不同地区、不同年龄的学习者都能无障碍获取知识
- [x] 更便捷 — 针对大型或不方便运输的机械以及特定区域的农作物，线上平台介绍更加高效

## <span class="c-blue">实践过程</span>

团队前往农村深入探索问题，发现农村人们大多以种植作物为主要收入来源，但对农作物生长只有经验上的了解，难以适应现代农业的快速发展。

{% alertBlockquote danger %}
农民往往会选择杂交品种以获得更优质的果实，却不知如何选择更优的种子；选择一种农药灭虫后一旦停用就无法抑制害虫反弹——诸如此类的 <span class="c-red">信息差问题</span> 让农民深受困扰。
{% endalertBlockquote %}

在确立目标后，团队开始对市面上可购买的农作物种子等进行 {% badge 测评 orange %}，收集用户反馈，发现作物产量与质量与农具的使用密切相关。在此基础上，除原有农作物测评外，还加入了 {% badge 农具教学视频 blue %} 等内容，不断提升平台实用性。
