## STEP 1 — <span class="c-red">Fork</span>

Visit the template repository and fork it. Go to Settings to configure GitHub Pages at `your-username.github.io`.

{% alertBlockquote tip SUCCESS %}
When your site loads at your-username.github.io, Step 1 is complete!
{% endalertBlockquote %}

## STEP 2 — Customize

Edit `_config.yml` to personalize your blog. Set up giscus for comments. Add Google Analytics.
